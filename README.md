## 🜸 *DESCRIPCIÓN*

**BALDWIND IV** es un bot de WhatsApp con temática **medieval/cyberpunk** diseñado para ofrecer una experiencia única. Con un sistema RPG completo, descargas multimedia, juegos y mucho más.

> *"El maná fluye a través del código, y el destino te aguarda en el Reino de BALDWIND"* — **DEVLYONN**

## 🛸 *CARACTERÍSTICAS PRINCIPALES*

### 📡 SISTEMA RPG
| Comando | Función |
|---------|---------|
| `#registrar Nombre.Edad` | Crea tu grimorio y únete al reino |
| `#perfil` | Ver estadísticas completas del guerrero |
| `#minar` | Extrae recursos y gana monedas |
| `#work` | Trabaja para ganar experiencia y monedas |
| `#daily` | Reclama tu recompensa diaria |
| `#balance` | Consulta tus monedas y banco |
| `#deposit <cantidad>` | Deposita monedas en el banco |
| `#withdraw <cantidad>` | Retira monedas del banco |
| `#shop` | Visita la tienda mágica |
| `#comprar <item>` | Adquiere items para mejorar tus stats |

### 🤖 DESCARGAS
| Comando | Función |
|---------|---------|
| `#yt <búsqueda>` | Busca y descarga video de YouTube |
| `#play <búsqueda>` | Busca y descarga audio de YouTube |
| `#tiktok <url>` | Descarga video de TikTok |
| `#ttsearch <búsqueda>` | Busca videos de TikTok |

### 👥 GRUPOS
| Comando | Función |
|---------|---------|
| `#add <número>` | Agrega un miembro al grupo |
| `#kick <@tag>` | Expulsa un miembro |
| `#promote <@tag>` | Ascender a administrador |
| `#demote <@tag>` | Descender de administrador |
| `#grupo` | Ver información del grupo |

### 🔧 ADMIN
| Comando | Función |
|---------|---------|
| `#ban <@tag>` | Banea a un usuario |
| `#unban <@tag>` | Desbanea a un usuario |
| `#delete` | Elimina el mensaje citado |
| `#setprefix <símbolo>` | Cambia el prefijo del bot |

### 🎮 JUEGOS
| Comando | Función |
|---------|---------|
| `#ppt <piedra/papel/tijera>` | Juega piedra, papel o tijera |
| `#dado` | Lanza un dado virtual |
| `#suerte` | Prueba tu fortuna |
